# 动画效果参考手册

生成 HTML 汇报时按需查阅此参考。根据预设的情感基调匹配动画。

## 情感-效果匹配指南
| 情感 | 动画特点 | 视觉线索 |
|------|----------|----------|
| **戏剧化 / 电影感** | 慢速淡入 (1-1.5s)，大范围缩放 (0.9→1)，视差滚动 | 深色背景、聚光效果、全幅图片 |
| **科技感 / 未来感** | 霓虹发光 (box-shadow)，故障/拼贴文字，网格揭露 | 粒子系统 (canvas)、网格图案、等宽字体点缀、青色/品红/电光蓝 |
| **活泼 / 友好** | 弹性缓动 (弹簧物理)，浮动/上下摆动 | 圆角、粉彩/明亮配色、手绘元素 |
| **专业 / 商务** | 微妙快速动画 (200-300ms)，干净的幻灯片切换 | 藏青/石板灰/炭灰、精确间距、数据可视化优先 |
| **沉稳 / 极简** | 非常缓慢的微妙运动，轻柔淡入 | 大量留白、柔和色调、衬线字体、充裕内边距 |
| **编辑风 / 杂志感** | 交错文字揭露、图文中穿插 | 强势字体层级、引用拉取、打破网格的布局、衬线标题 + 无衬线正文 |

## 入场动画
```css
/* 淡入 + 上滑（适用范围最广） */
.reveal {
    opacity: 0;
    transform: translateY(30px);
    transition: opacity 0.6s var(--ease-out-expo),                 transform 0.6s var(--ease-out-expo);
}
.visible .reveal {
    opacity: 1;
    transform: translateY(0);
}
/* 缩放进入 */
.reveal-scale {
    opacity: 0;
    transform: scale(0.9);
    transition: opacity 0.6s, transform 0.6s var(--ease-out-expo);
}
/* 从左滑入 */
.reveal-left {
    opacity: 0;
    transform: translateX(-50px);
    transition: opacity 0.6s, transform 0.6s var(--ease-out-expo);
}
/* 模糊渐显 */
.reveal-blur {
    opacity: 0;
    filter: blur(10px);
    transition: opacity 0.8s, filter 0.8s var(--ease-out-expo);
}
```

## 交互动效
```javascript
/* 3D 倾斜效果 — 为卡片/面板增加深度感 */
class TiltEffect {
    constructor(element) {
        this.element = element;
        this.element.style.transformStyle = 'preserve-3d';
        this.element.style.perspective = '1000px';
        this.element.addEventListener('mousemove', (e) => {
            const rect = this.element.getBoundingClientRect();
            const x = (e.clientX
            - rect.left) / rect.width - 0.5;
            const y = (e.clientY - rect.top) / rect.height - 0.5;
            this.element.style.transform = `rotateY($ {
                x * 10
            }
            deg) rotateX($ {
                -y * 10
            }
            deg)`;
        }
        );
        this.element.addEventListener('mouseleave', () => {
            this.element.style.transform = 'rotateY(0) rotateX(0)';
        }
        );
    }
}
```

## 故障排查
| 问题 | 解决方案 |
|------|----------|
| 字体不加载 | 检查 Fontshare/Google Fonts URL；确保 CSS 中的字体名称与实际引入名称匹配 |
| 动画不触发 | 验证 IntersectionObserver 是否在运行；检查 `.visible` class 是否被添加 |
| 滚动捕捉不工作 | 确保 html 上有 `scroll-snap-type: y mandatory`；每张幻灯片需有 `scroll-snap-align: start` |
| 移动端问题 | 在 768px 断点处禁用重量级效果；测试触摸事件；降低粒子数量 |
| 性能问题 | 谨慎使用 `will-change`；优先使用 `transform`/`opacity` 动画；对 scroll 处理器做节流 |
| 中文文字显示不佳 | 检查字体回退链是否正确；确保中文字体在英文字体之后声明；行高是否足够 |

## 精选动画 keyframes

以下 keyframes 经过生产验证，可直接用于各种交互场景。
```css
/* 屏幕截图入场动画 — 底部弹入 */
@keyframes screenRise {
    from {
        opacity: 0;
        transform: translateY(18px) scale(0.96);
    }
    to {
        opacity: 1;
        transform: translateY(0) scale(1);
    }
}
/* 屏幕光扫效果 — 高光从左到右扫过 */
@keyframes screenScan {
    0% {
        transform: translateX(-120%);
    }
    100% {
        transform: translateX(120%);
    }
}
/* 抽屉从右侧滑入 */
@keyframes drawerSlide {
    from {
        opacity: 0;
        transform: translateX(28px) scale(0.98);
    }
    to {
        opacity: 1;
        transform: translateX(0) scale(1);
    }
}
/* 证据气泡弹入 — 从下方弹入带弹性 */
@keyframes evidenceBubbleIn {
    from {
        opacity: 0;
        transform: translateX(-50%) translateY(18px) scale(0.92);
    }
    70% {
        opacity: 1;
        transform: translateX(-50%) translateY(-4px) scale(1.015);
    }
    to {
        opacity: 1;
        transform: translateX(-50%) translateY(0) scale(1);
    }
}
/* 脉冲反馈 — 缩放弹跳，用于按钮或开关 */
@keyframes pulseFeedback {
    0% {
        transform: scale(1);
    }
    45% {
        transform: scale(1.14);
    }
    100% {
        transform: scale(1);
    }
}
```
### 动画使用参考
| keyframe | 触发时机 | 适用元素 | 时长 | 缓动 |
|----------|----------|----------|------|------|
| `screenRise` | 元素入场/揭示 | 屏幕截图堆叠、卡片组 | 620ms | cubic-bezier(0.2, 0.82, 0.22, 1) |
| `screenScan` | screenRise 同时触发 | 截图上的高光伪元素 | 1100ms | ease-out |
| `drawerSlide` | 抽屉面板打开 | 侧边抽屉、文件浏览面板 | 260ms | cubic-bezier(.2,.8,.2,1) |
| `evidenceBubbleIn` | 气泡弹出 | 浮动提示、证据面板 | 260ms | cubic-bezier(.2,.9,.2,1) |
| `pulseFeedback` | 点击/状态切换 | 开关按钮、操作确认 | 940ms | ease |
|
