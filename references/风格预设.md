# 12套风格预设参考手册

> **渐进式加载说明**：Phase 2 默认不要整份读取本文件。请先读取 [style-presets/index.json](style-presets/index.json) 获取 12 套预设摘要和文件路由，再按候选读取 `style-presets/` 下的单个预设文件。本文件保留为旧流程兼容入口，以及字体回退策略、CSS 注意事项的集中参考。

精选的视觉风格集合，专为中文工作汇报场景设计。每套风格都有完整的 CSS 变量定义。**仅使用抽象 CSS 形状 — 不使用插图。**
**视口适配 CSS：** 强制性的基础样式，参见 [视口适配.css](视口适配.css)。每个 HTML 都必须包含。

## 深色主题

### 1. 大胆信号
**调性：** 自信、大胆、现代、高冲击力
**布局：** 深色渐变底色上的彩色主卡片。编号左上角，导航右上角，标题左下角。
**字体搭配：**
- 展示字体：`Archivo Black` (900)
- 正文字体：`Space Grotesk` (400/500)
**配色：**
```css
:root {
    --bg-primary: #1a1a1a;
    --bg-gradient: linear-gradient(135deg, #1a1a1a 0%, #2d2d2d 50%, #1a1a1a 100%);
    --card-bg: #FF5722;
    --text-primary: #ffffff;
    --text-on-card: #1a1a1a;
}
```
**签名元素：**
- 鲜艳彩色主卡片作为视觉焦点（橙色、珊瑚色等鲜明强调色）
- 大号章节编号（01、02 等）
- 导航面包屑，通过不同透明度区分激活/未激活状态
- 基于网格的精确对齐布局

## 中文字体回退策略

所有字体声明必须在中文字体之前加英文回退链。推荐的中文字体回退：
```css
/* 无衬线风格 — 用于黑体/等线风格的页面 */
font-family: "Plus Jakarta Sans", "Noto Sans SC", "Source Han Sans SC", "Microsoft YaHei", sans-serif;
/* 衬线风格 — 用于宋体/编辑风格的页面 */
font-family: "Cormorant", "Noto Serif SC", "Source Han Serif SC", "SimSun", serif;
/* 等宽风格 — 用于代码/终端风格的页面 */
font-family: "JetBrains Mono", "Noto Sans Mono SC", "Microsoft YaHei", monospace;
```
**要点：**
- 英文字体在前，中文字体在后
- `Noto Sans SC` 和 `Source Han Sans SC` 是目前最广泛的中文网页字体
- 如果用户需要离线环境查看，使用 `Microsoft YaHei` 作为 Windows 系统的安全回退
- 中文行高推荐设置为 1.6-1.8（比英文 1.4-1.5 略大）

## CSS 注意事项

### CSS 函数的负值
**错误写法 — 浏览器静默忽略（无控制台错误）：**
```css
right: -clamp(28px, 3.5vw, 44px);
/* 浏览器忽略此声明 */
margin-left: -min(10vw, 100px);
/* 浏览器忽略此声明 */
```
**正确写法 — 用 calc() 包裹：**
```css
right: calc(-1 * clamp(28px, 3.5vw, 44px));
/* 有效 */
margin-left: calc(-1 * min(10vw, 100px));
/* 有效 */
```
CSS 不允许在函数名前直接加 `-`。浏览器会静默丢弃整行声明 — 不报错，元素只是出现在错误的位置。**始终使用 `calc(-1 * ...)` 来对 CSS 函数值取负。**
