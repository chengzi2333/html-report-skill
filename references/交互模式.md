# 交互模式参考手册

定义可复用的交互模式。Phase 3 生成时根据用户选择的交互复杂度等级按需读取。

## 模式 1：模态弹窗系统  **适用所有需要"点击展开→查看详情→关闭"的场景。**

### HTML 骨架
```html
<div class="modal" id="exampleModal" role="dialog" aria-modal="true" hidden>
<div class="modal-backdrop" data-modal-close></div>
<div class="modal-panel" role="document">
<button class="modal-close" data-modal-close aria-label="关闭">✕</button>
<!-- 可变内容区：卡片详情、表单、文件浏览等 -->
</div>
</div>
```
### CSS

### JavaScript
```javascript
let modalLastFocus = null;
const openModal = (modal) => {
    modalLastFocus = document.activeElement;
    modal.hidden = false;
    document.body.classList.add('modal-open');
    modal.querySelector('.modal-close')?.focus();
}
;
const closeModal = (modal) => {
    modal.hidden = true;
    document.body.classList.remove('modal-open');
    modalLastFocus?.focus();
}
;
// 绑定关闭方式：遮罩点击 / X按钮 / Esc document.querySelector('.modal').addEventListener('click', (e) => {
    if (e.target.closest('[data-modal-close]')) closeModal(e.currentTarget);
}
);
document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape') {
        document.querySelectorAll('.modal:not([hidden])').forEach(closeModal);
    }
}
);
```
**要点：**
- 遮盖层总是使用 `data-modal-close` 属性，统一关闭行为
- 关闭按钮必须有 `aria-label="关闭"`
- 弹窗内如果还有子面板（如抽屉），子面板关闭时只关子面板，不要关整个模态

## 模式 2：悬浮联动聚焦（`:has()` 分组联动）  **极低成本换取极大视觉提升。适用于任何多卡片并排的页面。**
```css
/* 只在支持 hover 的鼠标设备上生效 */
@media (hover: hover) and (pointer: fine) {
    /* 当网格容器内任意卡片被 hover，同组未 hover 的卡片统一降低视觉权重 */
    :where(.card-grid, .value-stack, .insights, .evidence-grid,          .risk-grid, .roadmap-grid, .method-grid, .outcome-strip,          .hero-metrics, .surface-grid, .timeline, .difficulty-grid):       has(.card:hover) .card:not(:hover) {
        opacity: 0.42;
        filter: grayscale(0.28) saturate(0.74);
    }
    /* 被 hover 的卡片突出显示 */
    .card:hover {
        transform: translateY(-8px) scale(1.035);
        border-color: var(--accent);
        box-shadow:       0 30px 76px rgba(24, 32, 40, 0.22),       0 0 0 2px rgba(var(--accent-rgb, 183, 25, 43), 0.16);
        z-index: 8;
    }
}
```
**要点：**
- 使用 `:where()` 降低选择器优先级，方便后续覆盖
- 必须包裹在 `@media (hover: hover) and (pointer: fine)` 内，触屏设备不生效
- 降权效果：opacity + grayscale + saturate 三重降权
- 不支持的浏览器只是没有联动效果，不影响功能

## 模式 3：数据标注提示（Tooltip）  **纯 CSS 实现的 hover 提示，适用于指标说明、术语解释、数据来源。**

### CSS
```css
[data-tip] {
    cursor: help;
}
/* 气泡主体 */
[data-tip]::after {
    content: attr(data-tip);
    position: absolute;
    left: 50%;
    bottom: calc(100% + 14px);
    z-index: 20;
    width: min(320px, 72vw);
    padding: 12px 14px;
    border-radius: 12px;
    color: #fff8eb;
    background: rgba(17, 23, 29, 0.94);
    border: 1px solid rgba(246, 223, 189, 0.22);
    box-shadow: 0 18px 46px rgba(0, 0, 0, 0.34);
    font-size: 12px;
    line-height: 1.58;
    opacity: 0;
    pointer-events: none;
    transform: translate(-50%, 8px) scale(0.98);
    transition: opacity 180ms ease, transform 180ms ease;
}
/* 三角箭头 */
[data-tip]::before {
    content: "";
    position: absolute;
    left: 50%;
    bottom: calc(100% + 5px);
    z-index: 21;
    width: 14px;
    height: 14px;
    background: rgba(17, 23, 29, 0.94);
    border-right: 1px solid rgba(246, 223, 189, 0.20);
    border-bottom: 1px solid rgba(246, 223, 189, 0.20);
    transform: translateX(-50%) rotate(45deg);
    opacity: 0;
    pointer-events: none;
    transition: opacity 180ms ease;
}
/* hover 显示 */
[data-tip]:hover::after, [data-tip]:hover::before {
    opacity: 1;
}
[data-tip]:hover::after {
    transform: translate(-50%, 0) scale(1);
}
```
### HTML 用法
```html
<div class="metric" data-tip="需要暂停一天核对库存，建议安排在周末低峰时段">
<b>24小时</b>
<span>数据对账周期</span>
</div>
```
**要点：**
- 气泡背景色和文字色应从主题变量派生，此处给出的是通用深色方案
- 如果元素上方空间不足（顶部元素），改为 `top: calc(100% + 14px)` 显示在下方
- 同一个元素可以用 `data-tip` 给卡片级提示，子元素单独用其他属性

### JavaScript
```javascript
const evidenceItems = [ {
    src: 'assets/evidence-01.png', alt: '截图1：说明文字'
}
, {
    src: 'assets/evidence-02.png', alt: '截图2：说明文字'
}
, {
    src: 'assets/evidence-03.png', alt: '截图3：说明文字'
}
];
let evidenceIndex = 0;
const renderEvidence = () => {
    const item = evidenceItems[evidenceIndex];
    evidenceBubble.classList.add('is-switching');
    setTimeout(() => {
        evidenceImage.src = item.src;
        evidenceImage.alt = item.alt;
        evidenceCount.textContent = `$ {
            evidenceIndex + 1
        }
        /$ {
            evidenceItems.length
        }
        `;
        evidenceBubble.classList.remove('is-switching');
    }
    , 120);
}
;
// 切换 evidencePrev.addEventListener('click', () => {
    evidenceIndex = (evidenceIndex - 1 + evidenceItems.length) % evidenceItems.length;
    renderEvidence();
}
);
evidenceNext.addEventListener('click', () => {
    evidenceIndex = (evidenceIndex + 1) % evidenceItems.length;
    renderEvidence();
}
);
// 打开/关闭 evidenceToggle.addEventListener('click', () => {
    if (!evidenceBubble.hidden) {
        evidenceBubble.hidden = true;
        return;
    }
    evidenceBubble.hidden = false;
    renderEvidence();
}
);
document.addEventListener('click', () => {
    if (!evidenceBubble.hidden) evidenceBubble.hidden = true;
}
);
evidenceBubble.addEventListener('click', (e) => e.stopPropagation());
```

## 模式 5：卡片内部编辑（弹窗内轻量编辑）  在弹窗中直接编辑内容，不依赖页面级编辑模式。

### 触发按钮
```html
<button class="edit-dot" title="编辑内容" aria-label="编辑此卡片内容"></button>
```
```css
.edit-dot {
    appearance: none;
    width: 14px;
    height: 14px;
    border-radius: 50%;
    border: 1px solid rgba(var(--accent-rgb), 0.34);
    background: rgba(var(--accent-rgb), 0.18);
    box-shadow: 0 0 0 7px rgba(var(--accent-rgb), 0.045);
    cursor: pointer;
    transition: transform 180ms ease, background 180ms ease, box-shadow 180ms ease;
}
.edit-dot:hover {
    transform: scale(1.18);
    background: var(--accent);
    box-shadow: 0 0 0 9px rgba(var(--accent-rgb), 0.10);
}
```
### 编辑状态切换
```javascript
let isEditing = false;
editDot.addEventListener('click', () => {
    isEditing = !isEditing;
    modal.classList.toggle('is-editing', isEditing);
    editableTargets.forEach(el => {
        el.contentEditable = isEditing ? 'true' : 'false';
        el.spellcheck = false;
    }
    );
    if (!isEditing) saveChanges();
    // 退出编辑时自动保存
}
);
// 保存：序列化编辑内容 → localStorage 或变量 const saveChanges = () => {
    const data = {
        title: titleEl.textContent,
        /* ... */
    }
    ;
    localStorage.setItem('modal-edits', JSON.stringify(data));
}
;
```
**要点：**
- 编辑开关用不起眼的小圆点（非编辑状态下存在感极低）
- 进入编辑状态后，可编辑区域出现虚线边框
- 退出编辑时自动保存，不打断用户
- 弹窗内的编辑操作不影响弹窗外的页面状态
