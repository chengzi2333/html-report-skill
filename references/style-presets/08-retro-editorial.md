# 复古编辑风

**调性：** 个性、自信、编辑感
**布局：** 奶油色背景上的居中内容，抽象几何形状作为点缀。
**字体搭配：**
- 展示字体：`Fraunces` (700/900) — 独特衬线体
- 正文字体：`Work Sans` (400/500)
**配色：**
```css
:root {
    --bg-cream: #f5f3ee;
    --text-primary: #1a1a1a;
    --text-secondary: #555;
    --accent-warm: #e8d4c0;
}
```
**签名元素：**
- 抽象几何形状（圆形轮廓 + 线条 + 点）
- 加粗边框的 CTA 盒子
- 有个性、对话感的文案风格
- **不使用插图 — 仅用几何 CSS 形状**
