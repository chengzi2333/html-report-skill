# 暗夜植物园

**调性：** 优雅、精致、艺术感、高级
**布局：** 深色背景上居中内容，角落放置抽象柔和形状。
**字体搭配：**
- 展示字体：`Cormorant` (400/600) — 优雅衬线体
- 正文字体：`IBM Plex Sans` (300/400)
**配色：**
```css
:root {
    --bg-primary: #0f0f0f;
    --text-primary: #e8e4df;
    --text-secondary: #9a9590;
    --accent-warm: #d4a574;
    --accent-pink: #e8b4b8;
    --accent-gold: #c9b896;
}
```
**签名元素：**
- 抽象柔和渐变圆形（模糊、叠加）
- 暖色调强调色（粉色、金色、陶土色）
- 细竖线装饰
- 斜体签名排版
- **不使用插图 — 仅用抽象 CSS 形状**
