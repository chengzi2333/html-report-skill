# 电光工作室

**调性：** 利落、干净、专业、高对比
**布局：** 上下分栏 — 上方白色，下方蓝色。角落放置品牌标识。
**字体搭配：**
- 展示字体：`Manrope` (800)
- 正文字体：`Manrope` (400/500)
**配色：**
```css
:root {
    --bg-dark: #0a0a0a;
    --bg-white: #ffffff;
    --accent-blue: #4361ee;
    --text-dark: #0a0a0a;
    --text-light: #ffffff;
}
```
**签名元素：**
- 双面板垂直分栏
- 面板边缘的强调色条
- 引语排版作为视觉主角
- 极简、自信的间距
