# 笔记本标签

**调性：** 编辑感、有条理、优雅、有质感
**布局：** 深色背景上的奶油色纸张卡片，右侧边缘放置彩色标签。
**字体搭配：**
- 展示字体：`Bodoni Moda` (400/700) — 经典编辑风
- 正文字体：`DM Sans` (400/500)
**配色：**
```css
:root {
    --bg-outer: #2d2d2d;
    --bg-page: #f8f6f1;
    --text-primary: #1a1a1a;
    --tab-1: #98d4bb;
    --tab-2: #c7b8ea;
    --tab-3: #f4b8c5;
    --tab-4: #a8d8ea;
    --tab-5: #ffe6a7;
}
```
**签名元素：**
- 纸张容器 + 微妙阴影
- 右侧边缘彩色段落标签（竖排文字）
- 左侧装订孔装饰
- 标签文字需随视口缩放：`font-size: clamp(0.5rem, 1vh, 0.7rem)`
