# Changelog

本项目的重要公开变更记录在此文件中。

## 1.1.0 - 2026-07-16

### Added

- 新增 `scripts/inline_images.py`，将本地路径、绝对路径、`file://`、HTTP/HTTPS、CSS `url()` 和 `srcset` 图片转换为 Base64 Data URI。
- 新增全局 Phase 进入、完成和回退提示，以及 C01–C08 结构化确认点注册表。
- 新增 13 套中文适配预设，总数达到 25 套。
- 新增第三方预设集中声明，记录真实来源、原名称、许可证和中文适配说明。

### Changed

- `validate.py` 新增阻塞级“图片可移植性”检查。
- 交付顺序调整为“生成 → 图片内联 → 静态验证 → 内容覆盖 → 浏览器验收”。
- 保持旧 12 套预设正文和 `index.json` 数组接口兼容，新预设使用轻量扩展字段。

### Security

- 远程图片下载不携带登录态，阻止回环、私网、链路本地地址和携带认证信息的 URL。
- 每次远程重定向都会重新验证目标地址；非法 MIME、不可识别图片和下载失败会阻塞交付。
- 单图超过 10MB、最终 HTML 超过 50MB 时必须经过用户确认。

### Verification

- 46 个自动化测试通过。
- 可信 Demo 静态验证 26/26、100 分，Markdown 内容覆盖检查通过。
- 旧 12 套兼容、25 套预设路由、中文名称和第三方许可证归因检查通过。

## 1.0.0 - 2026-06-08

- 首个公开版本，支持 Markdown/PPT 转换、中文排版、风格预览、内容覆盖率和静态质量门禁。
